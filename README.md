# Customer Support & Operations Intelligence

## 📖 Sobre o Projeto

O **Customer Support Operations Intelligence** é um projeto completo de Engenharia e Análise de Dados desenvolvido para simular um ambiente corporativo real de uma empresa SaaS de atendimento ao cliente.

Diferente de projetos que utilizam bases públicas já prontas, todo o ecossistema foi construído do zero em união a LLM ChatGPT 5.5 para um ganho de produtividade e auxilio a complexidade do projeto ( auxiliando na modelagem dimensional, geração sintética dos dados, criação das regras de negócio, inclusão de inconsistências na base de dados, pipeline de tratamento, armazenamento em arquitetura Medallion (Bronze, Silver e Gold)) consultas SQL e visualizações analíticas no Power BI.

O objetivo é reproduzir desafios encontrados em operações reais de Customer Support, permitindo analisar indicadores operacionais, qualidade do atendimento, cumprimento de SLA, produtividade dos agentes, satisfação dos clientes (CSAT), backlog de chamados e eficiência dos processos de suporte.

Além da análise de negócio, o projeto também demonstra práticas de Engenharia de Dados, como validação da qualidade dos dados, tratamento de inconsistências, rastreabilidade das transformações (Data Lineage), organização em camadas e documentação técnica completa.


## 🎯 Objetivos do projeto

Este projeto foi desenvolvido com o propósito de demonstrar conhecimentos práticos em:

- Engenharia de Dados
- Business Intelligence
- Organização de Documentação
- Modelagem Dimensional
- Extração de Dados
- Transformação de Dados
- Carregamento de Dados
- Análise de Dados
- Qualidade de Dados
- Storytelling com Dados
- Construção de Dashboards Executivos

Além disso, busca simular um fluxo completo de dados, desde a geração dos registros operacionais até a disponibilização de informações para tomada de decisão.


## 💼 O que este projeto demonstra

Ao longo do projeto são aplicados conceitos utilizados em ambientes corporativos, como:

- Arquitetura Medallion (Bronze, Silver e Gold)
- Modelagem Star Schema
- Pipeline ETL utilizando Python
- Validação e tratamento de qualidade dos dados
- Injeção de inconsistências para simulação de cenários reais
- Organização dos dados em camadas analíticas
- Banco de dados PostgreSQL
- Consultas SQL para análise operacional
- Dashboards interativos no Power BI
- Documentação técnica completa
